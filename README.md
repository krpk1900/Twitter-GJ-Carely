# サービス概要

感謝を伝えたい人と[GJ Carely公式アカウント](https://twitter.com/GJ_Carely)を同時にメンションして感謝の言葉を伝えると返事が返ってきます。  
感謝を伝えた側、伝えられた側がもしiCARE社員であれば、社内で運用されているSlack版GJ Carelyのポイントとして加算されます。
